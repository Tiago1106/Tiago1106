# 👨🏽‍💻 Tiago de Freitas Pinheiro

**`Desenvolvedor Front End`**

Me chamo Tiago de Freitas Pinheiro, tenho 24 anos, sou de Franca/SP e formado em Engenharia de Software pela Unifacef. Atuo como desenvolvedor Front End com foco em criar interfaces modernas, performáticas e acessíveis.

Sou apaixonado por tecnologia e estou em constante evolução, sempre buscando novos conhecimentos e aprimorando minhas habilidades. Tenho experiência com React.js, Next.js, TypeScript, TailwindCSS e React Native/Expo no desenvolvimento de interfaces web e mobile. No Back End, utilizo Node.js, NestJS, Prisma e PostgreSQL para construção de APIs escaláveis.

Trabalho com responsabilidade, atenção aos detalhes e foco na entrega de valor real para o usuário. Acredito na importância de escrever um código limpo, bem estruturado e fácil de manter, além de colaborar de forma eficiente em equipe.
